# 貢獻者授權協議（CLA）

本貢獻者授權協議（「CLA」）適用於所有被接受到
**Arona** 專案（「本專案」）的貢獻，包括 Celestia Island
組織下所有採用本 CLA 的倉庫。本專案依據
Business Source License 1.1 授權，變更日期為 2030-01-01，
之後轉換為接收方可選擇的 Apache-2.0 或 MIT（請參閱
`LICENSE`）。

**在您的貢獻被合併之前，您必須接受本 CLA。** 接受
CLA 是每位貢獻者（以及每位雇主，若您代表公司貢獻）
一次性完成的操作。

> CLA 的存在僅出於兩個原因：(1) 確認您有權貢獻您所提交的內容，
> 以及 (2) 讓本專案保持其 BUSL→Apache/MIT 的授權路徑，
> 並在未來調整授權，而無需追蹤每一位過往的貢獻者。
> 它**不**會降低審查門檻，也**不**會要求本專案必須接受任何貢獻。

## 1. 簽署方式

- **個人貢獻者。** 在您的每個提交中新增 `Signed-off-by` 行

（對 `git commit` 傳遞 `-s`）。該簽署即為您對本 CLA 的電子接受，
等同於開發者原創聲明（DCO）加上下方第 3 節的授權：

  ```text
  Signed-off-by: Jane Doe <jane@example.com>
  ```

- **企業貢獻者。** 若您代表雇主貢獻，雇主還必須簽署

第 6 節中的企業 CLA，並將其提交給授權人，
電子郵件為 contact@celestia.world（或 `LICENSE` 中列出的聯絡方式）。
只有在雇主的企業 CLA 已存檔後，員工的 `Signed-off-by` 才有效。

## 2. 定義

- 「**您**」/「**貢獻者**」指接受本 CLA 的個人或實體。
- 「**貢獻**」指任何原始創作作品，包括對現有作品的任何修改或新增，

由您有意地提交給本專案以供納入。「有意地提交」包括
您透過 pull request、提交、issue、審查評論或同等管道發送的程式碼
和文件，但不包括您明確標記為「非貢獻」的作品。

- 「**授權人**」/「**維護者**」指本專案的版權持有人

（Celestia Island / langyo）以及被指定代表其行事的人員。

## 3. 版權授權

受本 CLA 約束，您授予授權人一個永久、全球、
非獨佔、免版稅、不可撤銷的版權授權，以複製、
製作衍生作品、公開展示、公開執行、再授權、
散佈及以其他方式利用您的貢獻，包括將貢獻
（單獨或與本專案一起）以 Business Source License 1.1、
Apache License 2.0、MIT License 或授權人選擇的任何其他授權
重新授權的權利，以及對下游接收方進行相同授權的權利。

## 4. 專利授權

受本 CLA 約束，您授予授權人及由授權人散佈的軟體接收方
一個永久、全球、非獨佔、免版稅、不可撤銷（除本節所述情況外）
的專利授權，以製造、已製造、使用、要約出售、出售、進口
及以其他方式轉移該貢獻，其中授權僅適用於您可授權的
那些專利請求項，這些請求項必然被您的貢獻單獨侵犯，
或被您的貢獻與提交貢獻時存在的本專案的組合侵犯。
若任何實體針對您或任何其他實體（包括反訴或交叉請求）
提起專利訴訟，主張該貢獻或其所提交的專案構成直接或
輔助專利侵權，則依據本 CLA 就該貢獻或專案授予該實體的
任何專利授權自訴訟提起之日起終止。

## 5. 您的聲明

您聲明：

(a) 您創作了該貢獻，或以其他方式有權提交該貢獻；

(b) 每個貢獻均為您的原創作品（或者，若代表第三方提交，
您有權代表其提交，並已確保獲得本 CLA 所要求的權利）；

(c) 據您所知，該貢獻未侵犯任何第三方的智慧財產權；以及

(d) 您依法有權授予第 3 節和第 4 節中的授權，且若
您的雇主對您創作之智慧財產享有權利，您的雇主
已就您的貢獻取得豁免或同意，或已簽署第 6 節中的企業 CLA。

## 6. 企業 CLA

若您代表公司貢獻，該公司必須簽署以下企業 CLA
區塊並將其傳送至 contact@celestia.world，以確認以下事項：

```text

地址：______________________________________________________
聯絡人姓名與電子郵件：_________________________________________
GitHub 組織：________________________________________________

本公司特此確認，代表其向本專案提交貢獻的員工，
接受本 CLA，並就該等貢獻授予第 3 節和第 4 節中的授權。

授權簽署人：_________________________________________________
職稱：________________________________________________________
日期：________________________________________________________
```

## 7. 免責聲明

除第 5 節中明示的保證外，貢獻按「現狀」提供，
不附帶任何形式的保證。授權人無義務為任何貢獻提供
任何支援或合併之。合併始終由維護者自行決定，
並受 `CONTRIBUTING.md` 中的貢獻政策約束，該政策不受本 CLA
及 2030-01-01 的授權轉換影響。
