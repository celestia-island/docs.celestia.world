# Shittim Chest (什亭之匣)

**Версия 0.1.0**

Shittim Chest — это пользовательская оболочка для платформы мультиагентного взаимодействия [entelecheia](https://github.com/celestia-island/entelecheia), построенная на Rust и Vue 3.

## Архитектура

Shittim Chest состоит из нескольких компонентов, работающих вместе для обеспечения полного пользовательского опыта:

- **arona** — Пользовательский интерфейс чата, который вы используете в данный момент, с потоковыми ответами, генерацией изображений, мониторингом статуса агентов, окном размышлений, просмотром удалённых устройств и поддержкой нескольких языков.
- **`shittim_chest`** — Унифицированный бэкенд на Rust + Axum, обрабатывающий аутентификацию (JWT + OAuth), независимую маршрутизацию LLM, API чата, генерацию изображений, вход вебхуков, прокси scepter и сигнализацию удалённых устройств.

## Связь с Entelecheia

[entelecheia](https://github.com/celestia-island/entelecheia) — это ядро мультиагентной оркестрации. Оно предоставляет среду выполнения агентов (scepter, 13 специализированных агентов, среду выполнения Cosmos/IEPL). Shittim Chest обрабатывает всё, с чем пользователь непосредственно взаимодействует — идентификацию, представление и коммуникацию.

Эти два проекта разделены намеренно: entelecheia управляет оркестрацией агентов, а shittim-chest управляет идентификацией и представлением пользователя. Они общаются через JWT-аутентифицированный HTTP/WebSocket. Учётные данные для входа хранятся в `shittim_chest_db`; данные о правах и идентификации хранятся в entelecheia_db. Это разделение позволяет оболочке фронтенда развиваться независимо от ядра агентов.

## Связь с Hikari

[hikari](https://github.com/celestia-island/hikari) — это шлюз и слой маршрутизации для экосистемы Celestia Island. Он служит точкой входа для всего внешнего трафика, обрабатывая маршрутизацию запросов, балансировку нагрузки и функциональность API-шлюза между shittim-chest, entelecheia и другими сервисами.

## Связь с Tairitsu

[tairitsu](https://github.com/celestia-island/tairitsu) — это кроссплатформенный фреймворк нативных приложений для экосистемы Celestia Island. Он предоставляет настольные и мобильные клиенты на основе Tauri, которые оборачивают arona как нативное приложение, вместе с инфраструктурой автоматизации браузера и тестирования, обеспечивающей рабочий процесс разработки.
