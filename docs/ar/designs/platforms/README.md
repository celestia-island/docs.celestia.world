# وثائق تصميم المنصة

> **النطاق.** هذه الوثائق *على مستوى المنصة*: تعبر
> `core` (entelecheia)، `webui` (shittim-chest) و `router`
> (evernight). تصاميم كل مشروع تعيش تحت فئاتها الفرعية الخاصة.

| الوثيقة | الملخص |
