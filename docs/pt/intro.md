# Bem-vindo à documentação do celestia-island

Este é o hub centralizado de documentação para o ecossistema **celestia-island**.

## Instalações

Cada instalação mantém seu próprio site de documentação, construído a partir do mesmo repositório que o código-fonte:

| Instalação | Descrição | Documentação |
| --- | --- | --- |
| **Entelecheia** | Plataforma de colaboração multi-agente | [entelecheia.docs.celestia.world](https://entelecheia.docs.celestia.world) |
| **Shittim Chest** | WebUI, backend, CLI shell | [shittim-chest.docs.celestia.world](https://shittim-chest.docs.celestia.world) |
| **Evernight** | Broker de controle remoto e protocolos industriais | [evernight.docs.celestia.world](https://evernight.docs.celestia.world) |

## Projetos da plataforma

| Grupo | Projeto | Descrição |
| --- | --- | --- |
| **Core** | [entelecheia](https://github.com/celestia-island/entelecheia) | Plataforma de colaboração multi-agente |
| **WebUI** | [shittim-chest](https://github.com/celestia-island/shittim-chest) | Shell voltado para o usuário |
| **Platforms** | [arona](https://github.com/celestia-island/plana) · [evernight](https://github.com/celestia-island/evernight) | Tipos de protocolos · Controle remoto |

## Bibliotecas de ferramentas

Bibliotecas Rust independentes que pertencem ao ecossistema celestia-island,
mas não estão vinculadas à plataforma entelecheia. Cada ferramenta possui
seu próprio site de documentação em `<name>.docs.celestia.world`.

| Biblioteca | Descrição | Docs |
| --- | --- | --- |
| [yuuka](https://github.com/celestia-island/yuuka) | Macro procedural para estruturas aninhadas complexas | [yuuka.docs.celestia.world](https://yuuka.docs.celestia.world) |
| [kirino](https://github.com/celestia-island/kirino) | Autenticação de confiança zero e RBAC | [kirino.docs.celestia.world](https://kirino.docs.celestia.world) |
| [noa](https://github.com/celestia-island/noa) | Controle de versão distribuído nativo em IA | [noa.docs.celestia.world](https://noa.docs.celestia.world) |
| [malkuth](https://github.com/celestia-island/malkuth) | Kit de supervisão de serviços | [malkuth.docs.celestia.world](https://malkuth.docs.celestia.world) |
| [seia](https://github.com/celestia-island/seia) | Busca web multi-motor | [seia.docs.celestia.world](https://seia.docs.celestia.world) |

## Organização

- **Meta** — políticas compartilhadas: licença, CLA, código de conduta, segurança
- **Guides** — guias práticos para começar e desenvolvimento
- **Designs** — documentos de arquitetura e design

Use o seletor de idioma (canto inferior direito) para alternar entre os idiomas disponíveis.
