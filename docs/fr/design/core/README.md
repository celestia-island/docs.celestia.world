# Documents de Conception

> **IMPORTANT** : Ce sont des documents de conception ambitieux. Ils décrivent l'architecture
> et l'ensemble des fonctionnalités prévues — PAS l'état actuel de l'implémentation.
> Pour une évaluation honnête de ce qui est réellement implémenté, consultez le
> [ARCHITECTURE.md](../../ARCHITECTURE.md) à la racine du dépôt, en particulier le
> tableau **Vérification de la Réalité des Composants** et la section **Lacunes Actuelles**.
> 2 des 12+ agents de domaine de Couche 2 prévus sont actifs dans l'espace de travail
> (Automatisation Web ; Ingénierie Logicielle Classique — analyse statique, revue de code,
> métriques de qualité, refactoring, LSP). Les conceptions restantes sont conservées pour
> une implémentation future.
