# 歡迎來到 celestia-island 文件

這是 **celestia-island** 生態系統的集中式文件中心。

## 平台專案

| 組別 | 專案 | 描述 |
| --- | --- | --- |
| **Core** | [entelecheia](https://github.com/celestia-island/entelecheia) | 多智能體協作平台 |
| **WebUI** | [shittim-chest](https://github.com/celestia-island/shittim-chest) | 面向使用者的 Shell |
| **Platforms** | [arona](https://github.com/celestia-island/arona) · [evernight](https://github.com/celestia-island/evernight) | 協議類型 · 遠端控制 |

## 工具庫

屬於 celestia-island 生態系統的獨立 Rust 庫，
但不與 entelecheia 平台綁定。每個工具都有自己的文件站點，
位址為 `<name>.docs.celestia.world`。

| 庫 | 描述 | 文件 |
| --- | --- | --- |
| [yuuka](https://github.com/celestia-island/yuuka) | 用於複雜巢狀結構的過程巨集 | [yuuka.docs.celestia.world](https://yuuka.docs.celestia.world) |
| [kirino](https://github.com/celestia-island/kirino) | 零信任驗證與 RBAC | [kirino.docs.celestia.world](https://kirino.docs.celestia.world) |
| [noa](https://github.com/celestia-island/noa) | AI 原生分散式版本控制 | [noa.docs.celestia.world](https://noa.docs.celestia.world) |
| [malkuth](https://github.com/celestia-island/malkuth) | 服務監控工具包 | [malkuth.docs.celestia.world](https://malkuth.docs.celestia.world) |
| [seia](https://github.com/celestia-island/seia) | 多引擎網頁搜尋 | [seia.docs.celestia.world](https://seia.docs.celestia.world) |
| [lagrange](https://github.com/celestia-island/lagrange) | 基於 Pest 的 Markdown 文件渲染器 | [lagrange.docs.celestia.world](https://lagrange.docs.celestia.world) |
| [ichika](https://github.com/celestia-island/ichika) | 執行緒池管線巨集 | [ichika.docs.celestia.world](https://ichika.docs.celestia.world) |
| [aoba](https://github.com/celestia-island/aoba) | Modbus 與資料來源 CLI | [aoba.docs.celestia.world](https://aoba.docs.celestia.world) |

## 組織

- **Meta** — 共享策略：授權條款、CLA、行為準則、安全
- **Guides** — 入門與開發實用指南
- **Designs** — 架構與設計文件
