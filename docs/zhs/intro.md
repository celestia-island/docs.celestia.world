# 欢迎来到 celestia-island 文档

这是 **celestia-island** 生态系统的集中式文档中心。

## 设施

每个设施维护自己的文档站点，与源代码构建自同一仓库：

| 设施 | 描述 | 文档 |
| --- | --- | --- |
| **Entelecheia** | 多智能体协作平台 | [entelecheia.docs.celestia.world](https://entelecheia.docs.celestia.world) |
| **Shittim Chest** | WebUI、后端、CLI shell | [shittim-chest.docs.celestia.world](https://shittim-chest.docs.celestia.world) |
| **Evernight** | 远程控制与工业协议代理 | [evernight.docs.celestia.world](https://evernight.docs.celestia.world) |

## 平台项目

| 组别 | 项目 | 描述 |
| --- | --- | --- |
| **Core** | [entelecheia](https://github.com/celestia-island/entelecheia) | 多智能体协作平台 |
| **WebUI** | [shittim-chest](https://github.com/celestia-island/shittim-chest) | 面向用户的 Shell |
| **Platforms** | [arona](https://github.com/celestia-island/arona) · [evernight](https://github.com/celestia-island/evernight) | 协议类型 · 远程控制 |

## 工具库

属于 celestia-island 生态系统的独立 Rust 库，
但不与 entelecheia 平台绑定。每个工具都有自己的文档站点，
地址为 `<name>.docs.celestia.world`。

| 库 | 描述 | 文档 |
| --- | --- | --- |
| [yuuka](https://github.com/celestia-island/yuuka) | 用于复杂嵌套结构的过程宏 | [yuuka.docs.celestia.world](https://yuuka.docs.celestia.world) |
| [kirino](https://github.com/celestia-island/kirino) | 零信任身份验证与 RBAC | [kirino.docs.celestia.world](https://kirino.docs.celestia.world) |
| [noa](https://github.com/celestia-island/noa) | AI 原生分布式版本控制 | [noa.docs.celestia.world](https://noa.docs.celestia.world) |
| [malkuth](https://github.com/celestia-island/malkuth) | 服务监控工具包 | [malkuth.docs.celestia.world](https://malkuth.docs.celestia.world) |
| [seia](https://github.com/celestia-island/seia) | 多引擎网页搜索 | [seia.docs.celestia.world](https://seia.docs.celestia.world) |
| [lagrange](https://github.com/celestia-island/lagrange) | 基于 Pest 的 Markdown 文档渲染器 | [lagrange.docs.celestia.world](https://lagrange.docs.celestia.world) |
| [ichika](https://github.com/celestia-island/ichika) | 线程池管道宏 | [ichika.docs.celestia.world](https://ichika.docs.celestia.world) |
| [aoba](https://github.com/celestia-island/aoba) | Modbus 与数据源 CLI | [aoba.docs.celestia.world](https://aoba.docs.celestia.world) |
