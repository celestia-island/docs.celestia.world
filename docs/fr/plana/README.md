> Fait partie de l'écosystème [celestia-island](https://github.com/celestia-island).

Aide les démons, agents et serveurs à effectuer une auto-mise à jour sans interruption et un équilibrage de charge multi-instances : cycle de vie signal/drain, sondes de santé, workers supervisés, transfert par socket-activation, verrous de coordination.
