# محلل إعدادات SSH

- **الحالة**: مقبول
- **التاريخ**: 2026-06-09
- **المؤلفون**: فريق Evernight الأساسي

## السياق

يتوقع المستخدمون أن يقرأ `evernight` ملف `~/.ssh/config` للحصول على أسماء مستعارة للمضيفين ومضيفات القفزة وملفات المفاتيح، تمامًا مثل أمر `ssh`. بدون ذلك، يجب على المستخدمين تكرار جميع معاملات الاتصال في كل استدعاء.

## القرار

تنفيذ محلل إعدادات SSH نقي بـ Rust يتعامل مع التوجيهات القياسية: `Host`، `HostName`، `User`، `Port`، `IdentityFile`، `ProxyJump`، `ForwardAgent`، `ServerAliveInterval`، إلخ. مطابقة أنماط glob لإدخالات `Host`. مدمج مع مجموعة الاتصالات للاتصالات الشفافة المبنية على الإعدادات.

## النتائج

### إيجابية

- توافق شامل مع SSH؛ دقة مضيف القفزة من الإعدادات.

### سلبية

- يجب تتبع تغييرات صيغة إعدادات OpenSSH.
- دعم توجيه `Match` معقد.
