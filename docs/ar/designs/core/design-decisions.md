# سجلات قرارات البنية المعمارية (ADR)

يسجل هذا الدليل القرارات المعمارية الرئيسية المتخذة أثناء تطوير Entelecheia. يشرح كل سجل قرار (ADR) **ما** تم اتخاذ قرار بشأنه، و**لماذا** تم اتخاذه، وما هي **المقايضات** التي تم النظر فيها.

تتبع سجلات القرارات [قالب مايكل نايجارد للقرارات المعمارية](https://cognitect.com/blog/2011/11/15/documenting-architecture-decisions). وهي غير قابلة للتغيير بمجرد نشرها — وتُعلّم القرارات التي تم تجاوزها وفقًا لذلك.
