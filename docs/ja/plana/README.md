デーモン、エージェント、サーバーがダウンタイムゼロの自己アップグレードとマルチインスタンス負荷分散を行えるよう支援します：シグナル/ドレインライフサイクル、ヘルスプローブ、スーパーバイズされたワーカー、ソケットアクティベーションハンドオフ、調整ロックを備えます。
