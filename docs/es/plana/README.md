> Part of the [celestia-island](https://github.com/celestia-island) ecosystem.

Ayuda a demonios, agentes y servidores a realizar autoactualización sin tiempo de inactividad y balanceo de carga de múltiples instancias: ciclo de vida signal/drain, sondas de salud, workers supervisados, traspaso por activación de socket y bloqueos de coordinación.
