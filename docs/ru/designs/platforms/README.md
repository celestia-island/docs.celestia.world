
# Платформенные проектные документы

> **Область.** Эти документы — *платформенного уровня*: они охватывают
> `core` (entelecheia), `webui` (shittim-chest) и `router` (evernight).
> Дизайны отдельных проектов находятся в их собственных подкатегориях.

## Индекс

| Документ | Краткое описание |
| --- | --- |
| [Супервизия, скользящее обновление и репликация](supervision-and-rolling-update.md) | Единая основа дерева супервизии, общая для всех трёх проектов: единые семантики сигналов/дрейна, socket activation в systemd для бесшовной передачи, подключаемый trait координационного замка, а также две стратегии отказоустойчивости (Replica = балансировка нагрузки ⊃ скользящее обновление; Leader/Follower = периферийная HA), построенные на одних и тех же примитивах Worker + Supervisor. |
