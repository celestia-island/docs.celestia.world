# Проектные Документы

> **ВАЖНО**: Это целевые проектные документы. Они описывают предполагаемую
> архитектуру и набор функций — НЕ текущее состояние реализации.
> Для честной оценки того, что реализовано на самом деле, см.
> [ARCHITECTURE.md](../../ARCHITECTURE.md) в корне репозитория, в частности
> таблицу **Проверка Компонентов** и раздел **Текущие Пробелы**.
> 2 из 12+ запланированных агентов домена Уровня 2 активны в рабочем пространстве
> (Веб-автоматизация; Классическая программная инженерия — статический анализ,
> проверка кода, метрики качества, рефакторинг, LSP). Остальные проекты сохранены
> для будущей реализации.
