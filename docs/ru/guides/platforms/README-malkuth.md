> Part of the [celestia-island](https://github.com/celestia-island) ecosystem.

Помогает демонам, агентам и серверам выполнять самообновление без простоев и балансировать нагрузку между несколькими экземплярами: жизненный цикл signal/drain, проверки работоспособности, контролируемые воркеры, передача управления через socket-activation, координационные блокировки.
