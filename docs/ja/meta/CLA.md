# コントリビューターライセンス同意書 (CLA)

本コントリビューターライセンス同意書（以下「CLA」）は、**Arona**プロジェクト（以下「本プロジェクト」）に受理されるすべての貢献、および本CLAを採用するCelestia Island組織配下の全リポジトリに適用されます。本プロジェクトはBusiness Source License 1.1の下でライセンスされており、変更日は2030年1月1日です。変更日以降は、受領者の選択によりApache-2.0またはMITに移行します（`LICENSE`を参照）。

**本CLAを承諾いただくまで、貢献をマージすることはできません。** CLAの承諾は、コントリビューターごと（および企業を代表して貢献する場合は雇用主ごと）に一度行っていただければ結構です。

> CLAが存在する理由は以下の2点のみです：(1) 提出された貢献について、提出する権利を有していることを確認すること、(2) 本プロジェクトがBUSL→Apache/MITのライセンス経路を維持し、過去の全コントリビューターを追跡することなく将来的にライセンスを変更できるようにすること。これはレビュー基準を**下げるものではなく**、本プロジェクトに貢献の受理を義務付けるものでもありません。

## 1. 署名方法

- **個人のコントリビューター。** 各コミットに`Signed-off-by`行を追加してください（`git commit`に`-s`を渡します）。この署名は、本CLAへの電子的な承諾を意味し、Developer Certificate of Origin (DCO)および下記第3条の許諾に相当します。

  ```text
  Signed-off-by: Jane Doe <jane@example.com>
  ```

- **法人のコントリビューター。** 雇用主を代表して貢献する場合、雇用主は第6条の法人CLAにも署名し、contact@celestia.world（または`LICENSE`記載の連絡先）宛てに許諾者へ提出する必要があります。従業員による`Signed-off-by`は、雇用主の法人CLAが提出された後にのみ有効となります。

## 2. 定義

- 「**あなた**」/「**コントリビューター**」とは、本CLAを承諾する個人または法人を意味します。
- 「**貢献**」とは、本プロジェクトに組み込むためにあなたが意図的に提出した、著作権の対象となるオリジナルの著作物を意味します。これには、既存の著作物の改変や追加も含まれます。「意図的に提出した」には、プルリクエスト、コミット、Issue、レビューコメント、またはこれらに準ずる手段を通じて送信したコードやドキュメントが含まれますが、あなたが明示的に「貢献ではない」と示した著作物は除外されます。
- 「**許諾者**」/「**メンテナー**」とは、本プロジェクトの著作権者（Celestia Island / langyo）およびその代理として行動する権限を付与された者を意味します。

## 3. 著作権ライセンスの許諾

本CLAに従い、あなたは許諾者に対し、あなたの貢献を複製、二次的著作物の作成、公衆送信、公の実演、サブライセンス、頒布、その他利用するための、永続的、全世界的、非独占的、無償、かつ取消不能の著作権ライセンスを許諾します。これには、貢献（単独または本プロジェクトと共に）をBusiness Source License 1.1、Apache License 2.0、MIT License、または許諾者が選択するその他のライセンスの下で再ライセンスする権利、および下流の受領者に対しても同様の権限を付与する権利が含まれます。

## 4. 特許ライセンスの許諾

本CLAに従い、あなたは許諾者および許諾者によって頒布されるソフトウェアの受領者に対し、貢献を製造、製造委託、使用、販売の申出、販売、輸入、その他移転するための、永続的、全世界的、非独占的、無償、かつ（本条に定める場合を除き）取消不能の特許ライセンスを許諾します。このライセンスは、あなたがライセンス可能な特許クレームのうち、あなたの貢献単独によって、または貢献提出時点の本プロジェクトとあなたの貢献との組み合わせによって必然的に侵害されるものにのみ適用されます。いずれかの法人が、あなたまたは他の法人に対して、貢献または貢献が提出された本プロジェクトが直接的または間接的な特許侵害を構成すると主張する特許訴訟（反訴または反論を含む）を提起した場合、当該法人に対して本CLAに基づき当該貢献または本プロジェクトについて付与された特許ライセンスは、訴訟が提起された日をもって終了します。

## 5. あなたの表明

あなたは以下を表明します：

(a) あなたが貢献を作成したか、または貢献を提出する権利を有していること；

(b) 各貢献があなたの独自の創作物であること（または、第三者を代表して提出する場合、あなたがその第三者を代表して提出する権利を有し、本CLAに基づき必要とされる権利を確保していること）；

(c) あなたの知る限り、貢献がいかなる第三者の知的財産権も侵害しないこと；

(d) あなたが第3条および第4条のライセンスを付与する法的権限を有しており、あなたの雇用主があなたの創作した知的財産に対して権利を有する場合、雇用主があなたの貢献について権利放棄もしくは同意を取得しているか、または第6条の法人CLAに署名していること。

## 6. 法人CLA

あなたが企業を代表して貢献する場合、当該企業は以下の法人CLA欄に署名し、contact@celestia.world宛てに返送することにより、以下を確認する必要があります。

## 7. 免責事項

第5条の明示的な保証を除き、貢献は「現状のまま」提供され、いかなる種類の保証も伴いません。許諾者は、いかなる貢献に対してもサポートを提供する義務、またはマージする義務を負いません。マージは常にメンテナーの単独の裁量に委ねられ、`CONTRIBUTING.md`の貢献ポリシーに従います。このポリシーは本CLAによっても、2030年1月1日のライセンス移行によっても変更されません。
