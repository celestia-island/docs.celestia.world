
# 설계 문서

> **중요**: 본 문서들은 열망적(aspirational) 설계 문서입니다. 의도된
> 아키텍처와 기능 세트를 기술하며, 현재 구현 상태를 나타내는 것이 아닙니다.
> 실제 구현된 사항에 대한 정직한 평가는 리포지토리 루트의
> [ARCHITECTURE.md](../../ARCHITECTURE.md), 특히
> **컴포넌트 현실 점검(Component Reality Check)** 표와 **현재 격차(Current Gaps)** 섹션을 참조하십시오.
> 12개 이상으로 계획된 Layer-2 도메인 에이전트 중 2개가 워크스페이스에서 활성 상태입니다
> (Web Automation; Classic Software Engineering — 정적 분석, 코드 리뷰,
> 품질 메트릭, 리팩터링, LSP). 나머지 설계는 향후 구현을 위해 보존됩니다.
