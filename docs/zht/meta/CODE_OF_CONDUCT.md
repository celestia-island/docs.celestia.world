# 行為準則

## 我們的承諾

我們作為成員、貢獻者和領袖，承諾讓社群中的參與
成為對所有人無騷擾的體驗，不論年齡、體型、
可見或不可見的身心障礙、族裔、性別特徵、性別
認同與表達、經驗水準、教育程度、社經地位、
國籍、個人外貌、種族、宗教或性別認同與性取向。

我們承諾以促進開放、歡迎、多元、包容且健康
社群的方式行動與互動。

## 我們的標準

有助於為社群營造積極環境的行為範例包括：

* 對他人展現同理心與善意
* 尊重不同的意見、觀點與經驗
* 給予並優雅地接受建設性回饋
* 承擔責任，向受我們錯誤影響的人道歉，

並從經驗中學習

* 專注於不僅對我們個人，也對整體社群最有利的事

不可接受的行為範例包括：

* 使用性暗示的語言或影像，以及任何形式的性關注或性試探
* 挑釁、侮辱或貶損的評論，以及人身或政治攻擊
* 公開或私下的騷擾
* 未經明確許可即公開他人的私人資訊，例如實體地址或電子郵件地址
* 其他在專業環境中可合理被視為不當的行為

## 執行責任

社群領袖負責闡明和執行我們可接受行為的標準，並將
對任何他們認為不當、具有威脅性、冒犯性或有害的行為
採取適當且公平的糾正措施。

社群領袖有權且有責任刪除、編輯或拒絕不符合本行為準則
的評論、提交、程式碼、wiki 編輯、issue 和其他貢獻，
並在適當情況下傳達審核決定的理由。

## 適用範圍

本行為準則適用於所有社群空間，也適用於個人正式
在公共空間代表社群時的情況。代表社群的範例包括
使用官方電子郵件地址、透過官方社交媒體帳號發文，
或在線上或線下活動中以指定代表身份行事。

## 執行

濫用、騷擾或其他不可接受行為的實例可報告給負責
執行的社群領袖，電子郵件為 contact@celestia.world。
所有投訴將迅速且公平地進行審查和調查。

所有社群領袖有義務尊重任何事件舉報者的隱私和安全。

## 執行指南

社群領袖在決定對任何他們認為違反本行為準則的行為
的後果時，將遵循以下社群影響指南：

### 1. 糾正

**社群影響**：使用不當語言或其他被認為不專業或
不受社群歡迎的行為。

**後果**：社群領袖發出私下的書面警告，說明違規的
性質以及該行為不當的原因。可能會要求公開道歉。

### 2. 警告

**社群影響**：透過單一事件或一系列行為造成的違規。

**後果**：發出警告，並對持續行為附帶後果。在指定
時間內不得與相關人員互動，包括主動與執行行為準則的
人員互動。這包括避免在社群空間以及社交媒體等外部
管道中的互動。違反這些條款可能導致暫時或永久封禁。

### 3. 暫時封禁

**社群影響**：嚴重違反社群標準，包括持續的不當行為。

**後果**：在指定時間內，禁止與社群進行任何形式的
互動或公開通訊。在此期間，不得與相關人員進行公開或
私下互動，包括主動與執行行為準則的人員互動。
違反這些條款可能導致永久封禁。

### 4. 永久封禁

**社群影響**：展現出違反社群標準的行為模式，包括
持續的不當行為、對個人的騷擾，或對某類個人的攻擊或貶損。

**後果**：永久禁止在社群內進行任何形式的公開互動。
